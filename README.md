# Bitcoin Tracker Frontend Task
We would like your help to fix and improve this mock application. This codebase is not what you would typically find at Moneysupermarket, it is a framework agnostic example created for the purpose of allowing candidates to demonstrate their skills in HTML, CSS & JavaScript.

The application should work with a modern browser and for the purpose of this task you should not worry too much about cross browser compatibility. Some tests have been provided with the application to help guide with development.

### Task
* Add a currency converter to the application using the supplied wireframe [README.wireframe](README.wireframe.png) as a guide for this feature. This URL can be used to perform the conversion: https://blockchain.info/tobtc?currency=GBP&value=20
  * **Considerations**
    * Add responsive behaviour to table and converter, using appropriate naming conventions / CSS approach
    * Accessibility of the form and content
    * Testing of functional layer (JS logic)
    * Form input should accept positive numbers only
    * Application performance

### Extension
* Extend the table to display all currencies in the response

## Getting Started

First install dependencies:

```sh
npm install
```

To run in hot module reloading mode:

```sh
npm start
```

## Testing

To run unit tests:

```sh
npm test
```

To run unit tests in watch mode:

```sh
npm run test:watch
```
